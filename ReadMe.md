What did you think of the project?

I thought this project was a great way for software engineers to challenge their
mindsets and at the same time come up their own creative solutions. 
Having autonomy over your code base is the best as it lets you use your
imagination to create products that can be useful. I feel as if this is how
it works in the real world.

What didn’t you like about the project?

I thought it could have been a bit more specific but at the same time
I do feel as if the user requirements seemed simple enough to come up with
a solution that is up to our own minds. 

How would you change the project or approach?

The main thing I would identify is how the data is inputted. Sometimes 
data could be from a file or from a list that is already generated. Depending on
the way it is could frame how a developer wants to code. Besides that I loved the whole
goal of this project.

Anything else you would like to share?
I feel like this was a fun at home task that helped me deep dive into
my own creative mind and I absolutely loved that.
